# 联调检查清单

## 1. 前后端联调顺序

建议顺序：

1. 先启动后端，确认数据库连接、migration、seed 正常
2. 用接口工具先测登录接口
3. 再测图片上传接口
4. 再测老客户识别接口
5. 再测新建订单接口
6. 再测订单列表和订单详情接口
7. 再测今日订单统计接口
8. 再测仓库待发货列表和确认发货接口
9. 最后测客户端订单详情接口
10. 前端页面全部串起来后，再做微信真机联调

## 2. 每个页面依赖哪些接口

### 登录页

- `POST /api/v1/auth/login`

### 运营端新建订单页

- `GET /api/v1/customers/identify`
- `POST /api/v1/uploads/images`
- `POST /api/v1/orders`

### 运营端订单列表页

- `GET /api/v1/orders`

### 运营端订单详情页

- `GET /api/v1/orders/:id`
- `PATCH /api/v1/orders/:id`

### 今日订单后台表页

- `GET /api/v1/stats/today-orders`
- `GET /api/v1/orders?todayOnly=true`

### 客户详情页

- `GET /api/v1/customers/identify?mobile=...`

### 仓库待发货订单列表页

- `GET /api/v1/warehouse/orders/pending`

### 仓库发货页

- `POST /api/v1/warehouse/orders/:id/ship`

### 客户本次订单查看页

- `GET /api/v1/public/orders/:orderNo?token=...`

## 3. 哪些地方最容易出 bug

### 登录与角色跳转

- token 存储后未正确跳转
- 角色判断不一致导致页面权限异常

### 图片上传

- 前后端域名不一致导致图片地址打不开
- 微信里上传后预览失败
- 文件大小或格式被后端拒绝

### 金额计算

- 商品小计、总额、邮费、优惠、实收金额不一致
- 前端浮点数精度问题导致后端校验失败

### 老客户识别

- 手机号格式不标准导致识别不到
- 历史收货资料回填字段错位

### 订单号与公开链接

- 示例 seed 与当天新建订单序号冲突
- 公开订单 token 错误时提示不清晰

### 仓库发货

- 待发货列表数据不全
- 确认发货后状态没刷新

## 4. 如何逐项测试

### 步骤 1：登录

- 使用 `operator / Operator@123`
- 使用 `warehouse / Warehouse@123`
- 确认登录后跳转到正确页面

### 步骤 2：老客户识别

- 输入 seed 中已有手机号，如 `13811112222`
- 确认能返回历史收货资料和历史订单
- 输入未存在手机号，确认返回“非老客户”

### 步骤 3：图片上传

- 上传 jpg/png/webp 图片
- 确认后端返回文件 ID 与 URL
- 确认前端能展示预览图

### 步骤 4：新建订单

- 填完整字段并提交
- 确认生成 `ZN` 前缀订单号
- 确认返回客户查看链接与 token
- 确认新订单状态为 `pending_shipment`

### 步骤 5：订单列表与详情

- 在列表中按状态筛选
- 按手机号或订单号搜索
- 点击进入详情页，确认商品图、金额、地址、备注显示正常

### 步骤 6：编辑订单

- 修改收件信息、商品信息、金额信息
- 提交后重新打开详情页
- 确认修改已生效

### 步骤 7：今日订单统计

- 打开今日订单后台表页
- 核对总数、总额、待发货、已发货、已取消是否和订单数据一致

### 步骤 8：仓库发货

- 使用仓库账号进入待发货列表
- 打开发货页，填写快递公司与运单号
- 点击确认发货
- 确认订单状态更新为 `shipped`

### 步骤 9：客户订单查看

- 使用订单号与 token 打开客户端链接
- 确认只展示本单
- 确认不展示内部备注、仓库备注、历史订单

## 5. 微信环境下需要重点测试什么

### 页面打开与路由

- 微信里直接打开首页是否正常
- 分享链接打开客户端订单页是否正常
- 浏览器返回是否会丢失页面状态

### 上传体验

- 微信选择图片是否正常
- 大图上传是否超时
- 上传后图片是否立即可访问

### 安全与链接

- 客户链接是否只能看到本单
- token 错误时是否正确提示
- 链接复制、转发后是否仍能正常访问

### 移动端布局

- iPhone 刘海屏安全区
- Android 长屏和小屏适配
- 底部按钮是否被微信底栏遮挡

### 性能

- 首次打开是否过慢
- 订单详情页多图是否卡顿
- 网络较差时接口提示是否足够明确
